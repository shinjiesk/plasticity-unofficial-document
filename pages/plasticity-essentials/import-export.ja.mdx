# インポート/エクスポート

P-メニューの「Import/Append」「Export」からインポート/エクスポートを行うことができます。<br />
3Dビューポートにファイルをドラッグ&ドロップすることでもインポートできます。

次のファイル形式をサポートしています。

## インポート [#file-import]

| ファイル形式                 | Indieライセンス | Studioライセンス |                                                                  |
| ---------------------------- | --------------- | ---------------- | ---------------------------------------------------------------- |
| plasticity (*.plasticity)    | ✓              | ✓               |                                                                  |
| STEP (*.stp; *.step)         | ✓              | ✓               |                                                                  |
| ParaSolid (*.x_t; *.x_b)     | ✓              | ✓               |                                                                  |
| Wavefront OBJ (*.obj)        | ✓              | ✓               |                                                                  |
| FBX (*.FBX)                  | ✓              | ✓               |                                                                  |
| STL (*.stl)                  | ✓              | ✓               |                                                                  |
| Image (*.png; *.jpg; *.jpeg) | ✓              | ✓               | 詳細は[こちら](importing-reference-images)を参照してください。 |
| IGES (*.igs; *.iges)         |                 | ✓               |                                                                  |
| SAT (*sat)                   |                 | ✓               |                                                                  |
| Rhino (*.3dm)                |                 | ✓               |                                                                  |
| DXF/DWG (*.dxf; *.dwg)        |                 | ✓               | Windowsのみサポート                                       |
