# Import/Export

You can perform import/export operations via the "Import/Append" and "Export" in the P-menu.<br />
You can also import by dragging and dropping files into the 3D viewport.

The following file formats are supported:

## Import [#file-import]

| File Format                  | Indie License | Studio License |                                                      |
| ---------------------------- | ------------- | -------------- | ---------------------------------------------------- |
| plasticity (*.plasticity)    | ✓            | ✓             |                                                      |
| STEP (*.stp; *.step)         | ✓            | ✓             |                                                      |
| ParaSolid (*.x_t; *.x_b)     | ✓            | ✓             |                                                      |
| Wavefront OBJ (*.obj)        | ✓            | ✓             |                                                      |
| FBX (*.FBX)                  | ✓            | ✓             |                                                      |
| STL (*.stl)                  | ✓            | ✓             |                                                      |
| Image (*.png; *.jpg; *.jpeg) | ✓            | ✓             | For details, see [here](importing-reference-images). |
| IGES (*.igs; *.iges)         |               | ✓             |                                                      |
| SAT (*sat)                   |               | ✓             |                                                      |
| Rhino (*.3dm)                |               | ✓             |                                                      |
| DXF/DWG (*.dxf; *.dwg)       |               | ✓             | Supported on Windows only                  |
