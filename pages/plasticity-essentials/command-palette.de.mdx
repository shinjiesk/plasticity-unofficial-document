# Befehlspalette
- Befehle, die keine Schaltflächen oder Tastenkombinationen haben, können von der Befehlspalette ausgeführt werden. Drücken Sie die <kbd>F</kbd> Taste im Arbeitsbereich, um die Befehlspalette anzuzeigen. Führen Sie einen Befehl aus, indem Sie seinen Namen eingeben.
- Außerdem können Sie jederzeit Tastenkombinationen für einen Befehl festlegen oder Ihre Lieblingsbefehle anheften.

## Befehl ausführen
1. Drücken Sie die <kbd>F</kbd> Taste.
2. Die Befehlspalette wird angezeigt.
3. Geben Sie den Namen des Befehls in das Eingabefeld ein.
4. Befehle, die den eingegebenen Worten entsprechen, werden angezeigt. Die Optionen werden beim Tippen eingegrenzt.
5. Verwenden Sie die <kbd>↑</kbd> und <kbd>↓</kbd> Pfeiltasten sowie die <kbd>Eingabe</kbd> Taste, um den gewünschten Befehl auszuwählen.
6. Der ausgewählte Befehl wird ausgeführt.
![Befehlspalette](/img/command-palette.gif)

## Tastenkombination zuweisen
1. Drücken Sie die <kbd>F</kbd> Taste.
2. Die Befehlspalette wird angezeigt.
3. Geben Sie den Namen des Befehls in das Eingabefeld ein.
4. Befehle, die den eingegebenen Worten entsprechen, werden angezeigt. Die Optionen werden beim Tippen eingegrenzt.
5. Klicken Sie mit der rechten Maustaste auf den gewünschten Befehl und wählen Sie **Assign Shortcut**.
![Befehlspalette Rechtsklick](/img/command-palette-right-click.png)
6. Wenn "Press a key to assign shortcut..." erscheint, drücken Sie die gewünschte Taste.
7. Dies weist eine Tastenkombination zu.
8. Um eine Tastenkombination zu entfernen, klicken Sie mit der rechten Maustaste auf den Namen des Befehls in der Befehlspalette und wählen Sie **Remove Shortcut**.

<Callout type="warning" emoji="⚠️">
Wenn Sie eine bereits verwendete Tastenkombination zuweisen, wird sie ohne Warnhinweis überschrieben. Um Duplikate zu vermeiden, können Sie Tasten wie <kbd>Alt</kbd>, <kbd>Strg</kbd>, <kbd>Umschalt</kbd> usw. kombinieren, wenn Sie Tastenkombinationen zuweisen.
</Callout>

## Zu Favoriten hinzufügen
1. Drücken Sie die <kbd>F</kbd> Taste.
2. Die Befehlspalette wird angezeigt.
3. Geben Sie den Namen des Befehls in das Eingabefeld ein.
4. Befehle, die den eingegebenen Worten entsprechen, werden angezeigt. Die Optionen werden beim Tippen eingegrenzt.
5. Klicken Sie mit der rechten Maustaste auf den gewünschten Befehl und wählen Sie **Add to favorites**.
![Befehlspalette Rechtsklick](/img/command-palette-right-click.png)
6. Dies fügt den Befehl Ihren Favoriten hinzu. Eine Schaltfläche wird unter dem Eingabefeld erscheinen.
7. Um einen Befehl aus Ihren Favoriten zu entfernen, klicken Sie mit der rechten Maustaste auf das Favoriten-Symbol oder den Namen des Befehls in der Befehlspalette und wählen Sie **Remove from Favorites**.
