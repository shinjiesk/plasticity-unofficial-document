# Ansichtsmodus

## Ansichtsmodus umschalten

Der Ansichtsmodus kann durch Klicken auf die Schaltfläche oben rechts auf dem Bildschirm oder durch Verwendung des Tastaturkürzels umgeschaltet werden.
![Render-Modus](/img/render-mode.png)

| Funktion                                                                                                 | Tastenkombination             | Schaltfläche | Befehlsname                          |
| -------------------------------------------------------------------------------------------------------- | ----------------------------- | ------------ | ------------------------------------ |
| Orthografische Kamera umschalten                                                                         | <kbd>Ziffernblock  5</kbd>             | (1)          | Viewport: toggle orthographic camera |
| Overlay-Anzeige umschalten<br/>(Dadurch wird die Rasteranzeige ein- und ausgeschaltet.)                  | <kbd>Alt + Umschalt + Z</kbd> | (2)          | Viewport: toggle overlays            |
| X-Ray-Modus umschalten<br/>(Hierdurch werden verborgene Linien sichtbar und auswählbar.)                 | <kbd>Alt + Z</kbd>            | (3)          | Viewport: toggle xray mode           |
| Render-Modus umschalten<br/>(Wenn aktiviert, spiegelt der Render-Modus die Materialeinstellungen wider.) |                               | (4)          |                                      |

## Ansichtsmodus auswählen

Wenn der Render-Modus ausgeschaltet ist, wählen Sie den Anzeigemodus durch Rechtsklick auf die Schaltfläche (4) und Auswahl aus den Miniaturansichten.
![Render-Modus Vorschau](/img/render-mode-preview.png)

- (5) Standard
- (6) Rot, glänzend
- (7) Überprüfung der Oberflächenkontinuität, horizontal
- (8) Überprüfung der Oberflächenkontinuität, vertikal
- (9) Highlights überprüfen
- (10) Matt
- (11) Silhouetten überprüfen
- (12) Keine Schatten

## Kanten und Flächen anzeigen umschalten

Das Umschalten der Anzeige von Kanten und Flächen kann durch Rechtsklick auf die Schaltfläche (4) und Auswahl aus den Miniaturansichten erfolgen.

| Funktion                                                                                     | Tastenkombination | Schaltfläche | Befehlsname            |
| -------------------------------------------------------------------------------------------- | ----------------- | ------------ | ---------------------- |
| Kantenanzeige umschalten<br />(Wenn dies deaktiviert ist, werden die Kanten ausgeblendet.)   |                   | (13)         | viewport: toggle Edges |
| Flächenanzeige umschalten<br />(Wenn dies deaktiviert ist, werden die Flächen ausgeblendet.) |                   | (14)         | viewport: toggle Faces |
