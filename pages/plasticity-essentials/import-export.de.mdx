# Import/Export

Sie können Import-/Exportvorgänge über die Optionen "Import/Append" und "Export" im P-Menü ausführen.<br />
Sie können auch Dateien importieren, indem Sie diese in den 3D-Ansichtsbereich ziehen und ablegen.

Die folgenden Dateiformate werden unterstützt:

## Importieren [#file-import]

| Dateiformat                 | Indie-Lizenz | Studio-Lizenz |                                                                  |
| --------------------------- | ------------ | ------------- | ---------------------------------------------------------------- |
| Plasticity (*.plasticity)   | ✓           | ✓            |                                                                  |
| STEP (*.stp; *.step)        | ✓           | ✓            |                                                                  |
| ParaSolid (*.x_t; *.x_b)    | ✓           | ✓            |                                                                  |
| Wavefront OBJ (*.obj)       | ✓           | ✓            |                                                                  |
| FBX (*.FBX)                 | ✓           | ✓            |                                                                  |
| STL (*.stl)                 | ✓           | ✓            |                                                                  |
| Bild (*.png; *.jpg; *.jpeg) | ✓           | ✓            | Weitere Details finden Sie [hier](importing-reference-images). |
| IGES (*.igs; *.iges)        |              | ✓            |                                                                  |
| SAT (*sat)                  |              | ✓            |                                                                  |
| Rhino (*.3dm)               |              | ✓            |                                                                  |
| DXF/DWG (*.dxf; *.dwg)       |              | ✓            | Nur auf Windows unterstützt                            |
