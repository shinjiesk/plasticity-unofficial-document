# Rectangular Array（長方形配列で複製）

<Reference shortcut="" command="Rectangular Array" button="コマンドバー右" />

カーブ、ソリッド、シートを長方形配列で複製できます

1. 複製したいオブジェクトを選択します。
2. コマンドバー右のボタン <ButtonIcon type="button-rectangular-array" /> をクリックします。
3. 複製プレビューが表示されます。
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y-axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z-axis:** Z軸方向に限定します。
4. 複製したい方向をクリックします。
5. コマンドダイアログとショートカットキーでオプションを設定します。
   - **Distance 1:** 距離を指定します。
   - **Number:** 複製する回数を指定します。
   - <kbd>Shift</kbd>-ホイール上回転 **Add copy:** 複製の数を増やします。
   - <kbd>Shift</kbd>-ホイール下回転 **Subtract copy:** 複製の数を減らします。
5. \<オプション\> もうひとつの方向にも配列で複製したいときは、ダイアログの**Select endpoint 2**をクリックします。
    1. 複製したい方向をクリックします。
    2. コマンドダイアログでオプションを設定します。
         - **Distance 2:** 距離を指定します。
         - **Repeat:** 複製する回数を指定します。
         - <kbd>Shift</kbd>-ホイール上回転 **Add copy:** 複製の数を増やします。
         - <kbd>Shift</kbd>-ホイール下回転 **Subtract copy:** 複製の数を減らします。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
