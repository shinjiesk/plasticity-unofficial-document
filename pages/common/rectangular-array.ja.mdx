# Rectangular Array（長方形配列で複製）

<Reference shortcut="" command="Rectangular Array" button="コマンドバー右" buttonIcon='button-rectangular-array' />

指定した数に応じて、線形のパターンでオブジェクトを複製できます。オプションで別の方向を定義して、矩形の配列を作成することもできます。

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. コマンドバー右のボタン <ButtonIcon type="button-rectangular-array" /> をクリックします。
3. 複製プレビューが表示されます。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
4. 複製したい方向をクリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Distance 1:** 距離を指定します。
      - **Number:** 複製する回数を指定します。
   - <Kst>キーボードショートカット</Kst>
   - <kbd>Shift</kbd> + ホイール上回転 **Add copy:** 複製の数を増やします。
   - <kbd>Shift</kbd> + ホイール下回転 **Subtract copy:** 複製の数を減らします。
5. \<オプション\> もうひとつの方向にも配列で複製したいときは、ダイアログの**Select endpoint 2**をクリックします。
   1. 複製したい方向を右クリックします。
   2. コマンドダイアログでオプションを設定します。
      - <Kst>コマンドダイアログ</Kst>
         - **Distance 2:** 距離を指定します。
         - **Repeat:** 複製する回数を指定します。
      - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift</kbd> + ホイール上回転 **Add copy:** 複製の数を増やします。
      - <kbd>Shift</kbd> + ホイール下回転 **Subtract copy:** 複製の数を減らします。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
