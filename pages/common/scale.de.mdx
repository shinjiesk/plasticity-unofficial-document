# Scale

<Reference shortcut="S" command="Scale" button="Links von der Befehlsleiste" />

Objekte mithilfe des Skalierungs-Gizmo skalieren. Optionen für die Einschränkung auf einer einzelnen Achse oder [**freestyle scale**](#freestyle-scale) sind verfügbar. Der Drehpunkt ist ebenfalls einstellbar.

1. Wählen Sie die Objekte aus, die Sie skalieren möchten.
2. Drücken Sie <kbd>S</kbd>.
3. Verwenden Sie den angezeigten Skalierungs-Gizmo zum Größenändern.
4. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - **XYZ:** Geben Sie die Skalierung der X-, Y- und Z-Achsen an.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Binden Sie an die X-, Y- oder Z-Achse](./sketching-essentials#constrain-to-an-axis).
   - <kbd>Shift + X</kbd>/<kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z plane:** Binden Sie an die X-, Y- oder Z-Ebene.
   - <kbd>S</kbd> **Uniform:** Skaliert gleichmäßig in den X-, Y- und Z-Richtungen.
   - <kbd>F</kbd> **Freestyle:** Geben Sie die Skalierungsachse durch Klicken auf den Anfangs- und Endpunkt an und klicken Sie dann mit dem gewünschten Skalierungsverhältnis.
   - <kbd>V</kbd> **Pivot:** Klicken Sie, um die Position des Gizmo anzugeben.
   - <kbd>W</kbd> **Worldspace:** Ändern Sie die Koordinatenachse des Gizmo in Weltkoordinaten.
5. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.

## Freestyle Scale[#freestyle-scale]

Dieser Befehl entspricht der Option <kbd>F</kbd> des Befehls **Scale**.

1. Wählen Sie die Objekte aus, die Sie skalieren möchten.
2. Verwenden Sie den Befehl **Freestyle Scale** in der Befehlspalette.
3. Geben Sie die Skalierungsachse durch Klicken auf den Anfangs- und Endpunkt an.
4. Bewegen Sie den Cursor, um das Objekt zu skalieren, und klicken Sie dann auf das gewünschte Skalierungsverhältnis.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - Die Option entspricht dem Befehl **Scale**.
6. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
