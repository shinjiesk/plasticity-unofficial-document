# Duplicate（複製）

<Reference shortcut="Shift + D" command="Duplicate" button="コマンドバー左"  buttonIcon='button-duplicate'/>

オブジェクトを複製し、Move Gizmoを使用して移動できます。単一軸の制約やフリースタイルの移動が可能です。ピボットは調整可能です。

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. キーボードショートカット<kbd>Shift + D</kbd>を押します。
3. 表示された移動ギズモを使って移動します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **XYZ:** X、Y、Z軸方向の移動量を指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z-plane:** X, Y, Z平面に拘束します。
      - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
      - <kbd>F</kbd> **Freestyle:** 始点と終点をクリックして、移動の方向と距離を指定します。
      - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
      - <kbd>W</kbd> **Worldspace:** ギズモの座標軸をワールド座標にします。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。

<Callout type="info">
   - 複数のリージョンを複製すると、結合されたシェイプとして複製されます。
</Callout>
