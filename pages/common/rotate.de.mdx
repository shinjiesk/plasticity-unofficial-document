# Rotate

<Reference shortcut="R" command="Rotate" button="Links von der Befehlsleiste"  buttonIcon='button-rotate'/>

Drehen Sie Objekte mit dem Rotate Gizmo. Optionen für Einschränkungen auf einer einzelnen Achse oder [**freestyle rotation**](#freestyle-rotate) sind verfügbar. Der Drehpunkt ist ebenfalls einstellbar.

## Grundlegende Nutzung

1. Wählen Sie die Objekte aus, die Sie drehen möchten.
2. Drücken Sie <kbd>R</kbd>.
3. Verwenden Sie den angezeigten Rotation Gizmo zum Drehen.
4. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
      - **Angle:** Geben Sie den zuletzt mit dem Rotation Gizmo angegebenen Winkel als numerischen Wert an.
      - **Axis:** Geben Sie die Drehachse mit dem Vektor X, Y oder Z an.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Binden Sie an die X-, Y- oder Z-Achse](/tool/sketching-essentials#constrain-to-an-axis).
   - <kbd>G</kbd> **Screen space:** Begrenzen Sie auf die sichtbare Ebene.
   - <kbd>F</kbd> **Freestyle:** Klicken Sie auf den Anfangs- und Endpunkt der Drehachse und dann auf den Referenzpunkt. Bewegen Sie den Cursor, um das Objekt zu drehen, und klicken Sie dann, um den gewünschten Winkel einzustellen.
   - <kbd>V</kbd> **Pivot:** Klicken Sie, um die Position des Gizmo anzugeben.
   - <kbd>W</kbd> **Worldspace:** Ändern Sie die Koordinatenachse des Gizmo in Weltkoordinaten.
5. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.

## Freestyle Rotate [#freestyle-rotate]

Dieser Befehl entspricht der Option <kbd>F</kbd> des Befehls **Rotate**.

1. Wählen Sie die Objekte aus, die Sie drehen möchten.
2. Verwenden Sie den Befehl **Freestyle Rotate** im Befehlspalette.
3. Klicken Sie auf den Anfangs- und Endpunkt der Drehachse und dann auf den Referenzpunkt.
4. Bewegen Sie den Cursor, um das Objekt zu drehen, und klicken Sie dann, um den gewünschten Winkel einzustellen.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - Die Option entspricht dem Befehl **Rotate**.
6. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
