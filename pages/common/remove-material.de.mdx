# Remove Material

<Reference shortcut="Alt + M" command="Remove Material" button="" />

Entfernen Sie das Material von ausgewählten Objekten und setzen Sie es auf das Standardmaterial zurück.

## Grundlegende Nutzung

1. Wählen Sie die Objekte aus, von denen Sie das Material entfernen möchten.
2. Drücken Sie <kbd>Alt + M</kbd>.
3. Das Material wird auf den Anfangswert zurückgesetzt.
