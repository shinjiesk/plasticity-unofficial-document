# Curve Array（カーブに沿って複製する）

<Reference shortcut="" command="Curve Array" button="" />

カーブ、ソリッド、シートをカーブに沿って複製できます。

1. 複製したいオブジェクトを選択します。
2. コマンドパレットで**Curve Array**を実行します。
3. 沿わせたいカーブを選択します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **Number:** 複製する数を指定します。
   - **Twist:** カーブに沿って捻る角度を度数で指定します。
   - **Scale:** カーブに沿って拡大縮小倍率を指定します。
   - **Alignment**
      - **Parallel:** 軌道に関わらず、平行に複製します。 
      - **Normal:** 軌道に沿った角度で複製します。
   - <kbd>Shift</kbd>-ホイール上回転 **Add copy:** 複製の数を増やします。
   - <kbd>Shift</kbd>-ホイール下回転 **Subtract copy:** 複製の数を減らします。
4. コマンドダイアログのOKまたは右クリックで確定します。
