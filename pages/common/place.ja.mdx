# Place（指定軸を基準に複製）

<Reference shortcut="Ctrl + D" command="Place" button="コマンドバー左"  buttonIcon='button-place'/>

参照点にオブジェクトを複製して配置できます。スケーリング、回転、軸の反転が可能です。ブーリアン操作も可能です。

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. キーボードショートカット<kbd>Ctrl + D</kbd>を押します。
3. 複製元の基準点をクリックします。
4. 複製先の基準点をクリックします。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>S</kbd> **Scale:** 拡大縮小ができます。
      - <kbd>A</kbd> **Angle:** 回転ができます。
      - <kbd>D</kbd> **Offset:** 連続して複製できます。
      - <kbd>F</kbd> **Flip:** 複製の軸を反転できます。
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **x/y/z-up:** 複製の軸をX/Y/Z軸方向にします。
      - <kbd>Shift + Q</kbd> **Slice:** ブーリアン対象との交点にソリッドを作成し、余りのソリッドも残ります。
      - <kbd>Q</kbd> **Union:** ブーリアン対象と結合します。
      - <kbd>W</kbd> **Difference:** ブーリアン対象から切り取ります。
      - <kbd>Shift + E</kbd> **Intersect:** ブーリアン対象との交点にソリッドを作成します。
      - <kbd>B</kbd> **New Body:** 新規にソリッドを作成します。ブーリアンは実行されません。
5. クリックで確定します。
6. さらに複製したいときは、ステップ４、５を繰り返します。
7. <kbd>Esc</kbd>で終了します。
