# Curve Array

<Reference shortcut="" command="Curve Array" button="" />

Objekte entlang einer Kurve duplizieren, indem die Anzahl der Duplikate angegeben wird. Verdrehung, Skalierung und Ausrichtung möglich.

## Grundlegende Nutzung

1. Wählen Sie das Objekt aus, das Sie duplizieren möchten.
2. Verwenden Sie den **Curve Array** Befehl in der Befehlspalette.
3. Wählen Sie die Kurve aus, die als Pfad verwendet werden soll.
4. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
   - **Number:** Geben Sie die Anzahl der Duplikationen an.
   - **Twist:** Geben Sie den Verdrehungswinkel entlang der Kurve in Grad an.
   - **Scale:** Geben Sie den Skalierungsfaktor entlang der Kurve an.
   - **Alignment:**
         - **Parallel:** Duplizieren Sie die Kurven parallel, unabhängig vom Pfad.
         - **Normal:** Duplizieren Sie die Kurven mit Winkeln, die an den Pfad ausgerichtet sind.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt + Rad hoch</kbd> **Add copy:** Erhöht die Anzahl der Duplikationen.
      - <kbd>Umschalt + Rad runter</kbd> **Subtract copy:** Verringert die Anzahl der Duplikationen.
4. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
