# Move

<Reference shortcut="G" command="Move" button="Left of Command Bar"  buttonIcon='button-move'/>

Move objects using Move Gizmo. Options for single-axis constraints or [**freestyle movement**](#freestyle-move) are available. The pivot is also adjustable.

## Basic Usage

1. Select the objects you want to move.
2. Press <kbd>G</kbd>.
3. Use the displayed Move Gizmo to move.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **XYZ:** Specify the amount of movement of the X, Y, and Z axes.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](/tool/sketching-essentials#constrain-to-an-axis).
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z plane:** Constrain to the X, Y, or Z-plane.
      - <kbd>G</kbd> **Screen space:** Limit to the visible plane.
      - <kbd>F</kbd> **Freestyle:** Click to specify the starting point and endpoint, which will determine the direction and distance of movement.
      - <kbd>V</kbd> **Pivot:** Click to specify the position of the Gizmo.
      - <kbd>W</kbd> **Worldspace:** Change the Gizmo's coordinate axis to world coordinates.
4. Confirm with OK in the Command Dialog or right-click.

<Callout type="info" emoji="ðŸ’¡">
**Tips**
- To move selected objects to the origin, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > right-click > right-click.
- To move an object to the origin based on a selected point, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > Click the object's point > right-click.
</Callout>

## Freestyle Move[#freestyle-move]

This command is equivalent to the <kbd>F</kbd> option of the **Move** command.

1. Select the objects you want to move.
2. Use the **Freestyle Move**  command in the Command Palette.
3. Click the starting point and endpoint to determine the direction and distance of movement.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - The option is the same as the **Move** command.
4. Confirm with OK in the Command Dialog or right-click.
