## Sketch Commands - Region, Curve, Vertex, Control Vertex

| Command Name                                                      | Shortcut             | Note                                 |
| ----------------------------------------------------------------- | -------------------- | ------------------------------------ |
| [Fillet](sketch/fillet)                                           | <kbd>B</kbd>         | Fillet or chamfer Curves             |
| [Offset Curve](sketch/offset-curve)                               | <kbd>O</kbd>         | Offset Curves                        |
| [Subdivide Curve](sketch/subdivide-curve)                         | <kbd>Shift + S</kbd> | Increase control vertices of Curves  |
| [Extend](sketch/extend)                                           |                      | Extend Curve                         |
| [Rebuild](sketch/rebuild)                                         |                      | Rebuild Curve                        |
| [Text](sketch/text)                                               |                      | Create Curves from text              |
| **Project Commands**                                              |                      |                                      |
| [Project](sketch/project)                                         | <kbd>I</kbd>         | Project Curve to Solid or Sheet      |
| [Duplicate Curve and Project](sketch/duplicate-curve-and-project) |                      | Duplicate Curve And Project          |
| **Join and Split Commands**                                       |                      |                                      |
| [Join](sketch/join)                                               | <kbd>J</kbd>         | Join Curves at connecting points     |
| [Bridge](sketch/bridge)                                           |                      | Join Curves                          |
| [Trim](sketch/trim)                                               | <kbd>T</kbd>         | Trim Curve                           |
| [Cut Curve](sketch/cut-curve)                                     |                      | Cut Curve                            |
| [Split Segment](sketch/split-segment)                             |                      | Split a Curve into two segments      |
| **Vertex Commands**                                               |                      |                                      |
| [Delete Redundant Topology](sketch/delete-redundant-topology)     |                      | Remove excess points                 |
| [Convert Vertex](sketch/convert-vertex)                           |                      | Convert Vertices to Control Vertices |
