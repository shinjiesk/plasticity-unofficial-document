# Text

<Reference shortcut="" command="Text" button="" />

Creates Curves from the inputted text. Built-in fonts and sizes can be specified.

## Basic Usage

1. Use the **Text** command in the Command Palette.
2. Set the options in the Command Dialog.
   - <Kst>Command Dialog</Kst>
      - **Text box:** Enter the text you want to create.
      - **Font:** Select the desired font.
      - **Size:** Specify the font size.
3. Confirm with OK in the Command Dialog or right-click.
4. The text Curves are created.

<Callout emoji="â—">
**Note**<br />
The default font size is 1cm in height and the text will be positioned at the origin of the XY plane.
</Callout>
