# Text（テキスト）

<Reference shortcut="" command="Text" button="" />

入力したテキストでカーブを作成できます。

1. コマンドパレットで **Text** を実行します。
2. コマンドダイアログでオプションを設定します。
   - **Text box:** スケッチしたいテキストを入力します。
   - **Font:**  好みのフォントを選択します。
   - **Size:** フォントのサイズを指定します。
3. コマンドダイアログのOKをクリックするか、右クリックで確定します。
4. テキストのカーブが作成されます。

<Callout emoji="❗">
 **注意**<br /> 
 デフォルトのフォントサイズは高さ1cmで、テキストはXY平面の原点に配置されます。
 </Callout>
