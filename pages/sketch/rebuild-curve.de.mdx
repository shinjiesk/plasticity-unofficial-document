# Rebuild Curve [#rebuild-curve]

<Reference shortcut="" command="Rebuild Curve" button="Rechts von der Befehlsleiste" buttonIcon='button-rebuild' />

Durch Optimieren der Positionen der Control Vertices(CVs) neu aufgebaut, ohne die Form des Curve zu ändern.

## Grundlegende Nutzung

1. Wählen Sie den Curve aus, den Sie neu aufbauen möchten.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-rebuild" />
3. Legen Sie die Optionen im Command Dialog fest.
  - <Kst>Befehlsdialog</Kst>
    - **Method:**
      - **Refit:** Neuaufbau innerhalb von Toleranzen.
        - **Tolerance:** Geben Sie die Toleranzen an.
        - **Keep corners:** Wenn aktiviert, werden die Ecken des Curve nicht geändert.
      - **# points** Neuaufbau innerhalb der angegebenen Anzahl von CVs.
        - **Points:** Geben Sie die Anzahl der CVs an.
4. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
