# Bridge Edge[#bridge-edge]

<Reference shortcut="" command="Bridge Edge" button="" />

Verbindet zwei Edges, die G0/G1/G2/G3-kontinuierlich sein können. Geben Sie an, welches Ende der Edges Sie verbinden möchten. Die Spannung ist einstellbar.

## Grundlegende Nutzung

1. Wählen Sie zwei Edges aus, die Sie verbinden möchten.
2. Verwenden Sie den **Bridge** Befehl in der Command Palette.
3. Legen Sie die Optionen im Command Dialog und den Tastaturkürzeln fest.
     - <Kst>Befehlsdialog</Kst>
          - **Side:** 
               - **Side 1:** Geben Sie an, welche Seite des Endpunkts des zuerst ausgewählten Edge verbunden werden soll.
               - **Side 2:** Geben Sie an, welche Seite des Endpunkts des anschließend ausgewählten Edge verbunden werden soll.
          - **T:**
               - **Value 1/2:** Die relative Position auf dem Segment des Startpunkts/Endpunkts.
          - **Direction:** 
               - **Direction 1/2:** Die Richtung des Startpunkts/Endpunkts umkehren.
          - **Tension 1:**
               - **Value 1/2:** Der Spannungswert betrifft den 1. Control-Vertex(CV) bis 3. CV des Startpunkts/Endpunkts.
          - **Tension 2:**
               - **Value 1/2:** Der Spannungswert betrifft den 2. CV und 3. CV des Startpunkts/Endpunkts.
          - **Tension 3:**
               - **Value 1/2:** Der Spannungswert betrifft den 3. CV des Startpunkts/Endpunkts.
          - **Start curvature:**
               - **G0, G1, G2, G3:** Spezifizieren Sie die [continuity of the  Curve](/cad-essentials/continuity-curve-and-surface) am Startpunkt.
          -  **End curvature:**
               - **G0, G1, G2, G3:** Spezifizieren Sie die [continuity of the  Curve](/cad-essentials/continuity-curve-and-surface) am Endpunkt.      
          - **Trim:** Wenn aktiviert, wird jede Curve, die über die Brücke hinausgeht, getrimmt.
4. Bestätigen Sie mit OK im Command Dialog oder mit einem Rechtsklick.
