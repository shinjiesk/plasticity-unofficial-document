# Create Outline

<Reference shortcut="" command="Create Outline" button="" />

ソリッドまたはシートの外形を法線方向に沿って現在の構築平面に投影し、カーブを生成します。**Project Outline** コマンドは2Dのカーブを生成しますが、このコマンドは3Dのカーブを生成します。カーブはオブジェクトの元の境界位置に正確に配置されます

## 基本的な使い方

1. 投影したい構築平面をセットします。
2. 投影するソリッドまたはシートを選択します。
3. コマンドパレットで **Create Outline** コマンドを実行します。
4. 投影されたカーブが作成されます。
5. 移動ギズモを使用してカーブを移動することもできます。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
