# Rebuild（再構築する）

<Reference shortcut="" command="Rebuild" button="Right of Command Bar" />

このコマンドは、選択されていたオブジェクトに応じて以下のコマンドを自動的に呼び出します。
  - カーブが選択されているとき: 以下の **Rebuild Curve** を参照してください。
  - フェイスが選択されているとき: Solid Commands セクションの [**Rebuild Face**](../solid/rebuild#rebuild-face) を参照してください。

## Rebuild Curve（カーブを再構築する）[#rebuild-curve]

カーブの形状は変えずに制御点の位置を最適化することで、カーブを再構築できます。

1. 再構築したいカーブを選択します。
2. ボタン <ButtonIcon type="button-rebuild" /> をクリックします。
3. コマンドダイアログでオプションを設定します。
  - **Method:**
    - **Refit:** 許容誤差の範囲内で再構築します。
      - **Tolerance:** 許容誤差を指定します。
    - **# points** 指定した制御点の数で再構築します。
      - **Points:** 制御点の数を指定します。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
