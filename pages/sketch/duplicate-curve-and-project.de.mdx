# Duplicate Curve and Project

<Reference shortcut="Alt + D" command="Duplicate Curve and Project" button="Links von der Befehlsleiste" />

Erstelle eine Kurve, die auf die Konstruktionsebene von einer Kurve oder Kante projiziert ist. Einzelachsenbeschränkungen oder freies Bewegen sind möglich.

1. Aktiviere die Konstruktionsebene, auf die du projizieren möchtest.
2. Wähle die Kurve oder Kante aus, die du duplizieren möchtest.
3. Drücke <kbd>Alt + D</kbd>.
4. Das Bewegungs-Gizmo erscheint und du kannst die projizierte Kurve frei bewegen.
   - Du kannst auch die Bewegungsrichtung einschränken.
      - <kbd>Umschalt + X</kbd>/ <kbd>Umschalt + Y</kbd>/<kbd>Umschalt + Z</kbd> **X, Y, Z Ebene:** Beschränke auf die X, Y oder Z-Ebene.
      - <kbd>G</kbd> **Screen space:** Beschränkung auf die sichtbare Ebene.
      - <kbd>F</kbd> **Freestyle:** Beschränke die Richtung durch zwei Punkte.
      - <kbd>V</kbd> **Pivot:** Beschränke die Richtung durch Klicken auf die Ebene.
      - <kbd>W</kbd> **Worldspace:** Löse die Richtungsbeschränkung auf.
5. Bestätige mit OK im Befehlsdialog oder Rechtsklick.
