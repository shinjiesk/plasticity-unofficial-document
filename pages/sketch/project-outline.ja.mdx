# Project Outline

<Reference shortcut="Alt + D" command="Project Outline" button="" />

ソリッドまたはシートの外形を現在の構築平面に法線の方向に投影し、カーブを生成します。

## 基本的な使い方

1. 投影したい構築平面をセットします。
2. 投影するソリッドまたはシートを選択します。
3. <kbd>Alt + D</kbd> を押してコマンドを実行します。
4. 投影された曲線が現在の構築平面に作成されます。
5. 移動ギズモを使用して曲線を移動することもできます。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
