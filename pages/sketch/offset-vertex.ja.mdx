# Offset Vertex

<Reference shortcut="O" command="Offset Vertex" button="" />

頂点の両側にカーブに沿って新しい頂点を挿入します。

## 基本的な使い方

1. 頂点を選択します。
2. <kbd>O</kbd> を押します。
3. マウスを動かして、新しい頂点の位置を決めます。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Distance:** 距離を指定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>D</kbd> **Distance:** マウスの動きで距離を指定します。
5. クリックして頂点の位置を固定します。
6. コマンドダイアログのOKか右クリックで確定します。
