# Fillet

<Reference shortcut="B" command="Fillet" button="Right of Command Bar"   buttonIcon='button-fillet'/>

Unified command for **Fillet Curve**, **Fillet Vertex**, **Fillet Shell**. The command is run based on the selected object.
   - When Curves are selected: see [**Fillet Curve**](fillet-curve).
   - When verticies are selected: see [**Fillet Vertex**](fillet-vertex).
   - When Edges are selected: see [**Fillet Shell**](/solid/fillet-shell) in the Solid Commands section.
