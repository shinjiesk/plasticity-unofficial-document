# Project Curve Curve [#project-curve-curve]

<Reference shortcut="I" command="Project Curve Curve" button="Right of the command bar"  buttonIcon='button-project'/>

Create a new Curve by projecting the intersection points of surfaces generated from two independent Curves.

## Basic Usage

1. Select the two Curves to be projected.
2. Press <kbd>I</kbd> or click the button.
3. The projected intersection Curve will be created.
