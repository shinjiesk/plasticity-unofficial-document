# Project Outline

<Reference shortcut="Alt + D" command="Project Outline" button="" />

Projizieren Sie die Kontur der Solids oder Sheets auf die aktuelle Bauebene entlang der normalen Richtung, wodurch eine Kurve entsteht.

## Grundlegende Nutzung

1. Legen Sie die gewünschte Bauebene für die Projektion fest.
2. Wählen Sie die Solids oder Sheets aus, die projiziert werden sollen.
3. Drücken Sie <kbd>Alt + D</kbd>, um den Befehl auszuführen.
4. Die projizierten Kurven werden auf der aktuellen Bauebene erstellt.
5. Optional können Sie die Kurven mit dem Move Gizmo verschieben.
6. Klicken Sie mit der rechten Maustaste oder klicken Sie im Befehlsdialog auf OK, um zu bestätigen.
