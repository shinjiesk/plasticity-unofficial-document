# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Right of Command Bar"  buttonIcon='button-offset-curve'/>

Unified command for Offset Vertex, Offset Planar Curve, Offset Region, Offset Face Loop, Offset Edge. The command is run based on the selected object.
   - When Vetices are selected: [Offset Vertex](offset-vertex).
   - When Curves are selected: [Offset Planar Curve](offset-planar-curve).
   - When Regions are selected: [Offset Region](offset-region).
   - When Faces are selected: see [Offset Face Loop](/solid/offset-face-loop) in the Solid Commands section.
   - When Edges are selected: see [Offset Edge](/solid/offset-edge) in the Solid Commands section.
