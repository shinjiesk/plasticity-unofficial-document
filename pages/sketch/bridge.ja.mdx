# Bridge

<Reference shortcut="" command="Bridge" button="" />

このコマンドは、選択されていたオブジェクトに応じて以下のコマンドを自動的に呼び出します。
- ふたつのカーブが選択されているとき、または何も選択されていないとき: 下記の [**Bridge Curve**](#bridge-curve) を参照してください。
- ふたつのポイントが選択されているとき: 下記の [**Bridge Vertex**](#bridge-vertex) を参照してください。
- ふたつのエッジが選択されているとき: 下記の [**Bridge Edge**](#bridge-edge) を参照してください。

## Bridge Curve（カーブをなめらかに繋げる）[#bridge-curve]

連続性を考慮してふたつのカーブを繋げることができます。また、エッジ上、フェース上、構築平面上の任意の点も繋げることができます。

<Callout emoji="❗">
    **注意**<br />
ポイントを選択する前に、[**グリッドスナップ**](../3d-viewport/grid-and-snap#grid-snap)と[**オブジェクトスナップ**](../3d-viewport/grid-and-snap#object-snap)を有効にしてください。
</Callout>
<Callout type="info" emoji="ℹ️">
始点と終点は、カーブ上、エッジ上、フェース上、または構築平面上の任意の点を選択することができます。
</Callout>

1. ツールバーの <ButtonIcon type="button-bridge-curve" /> をクリックします。
2. 始点をクリックします。
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y-axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z-axis:** Z軸方向に限定します。
   - <kbd>N</kbd> **Normal:** フェイスの法線方向に限定します。
3. 終点をクリックします。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **T:**
        - **Value 1:** 始点のセグメント上の相対位置(0~1.0)を指定します。
        - **Value 2:** 終点のセグメント上の相対位置(0~1.0)を指定します。
   - **Direction:** 
        - **Direction 1**: 始点の方向を反転します。
        - **Direction 2**: 終点の方向を反転します。
   - **Tension 1:**
        - **Value 1:** 始点のテンション値(\<0,0\<)を指定します。
        - **Value 2:** 終点のテンション値(\<0,0\<)を指定します。
   - **Tension 2:**
        - **Value 1:** 
        - **Value 2:**
   - **Tension 3:**
        - **Value 1:** 
        - **Value 2:**
   - **Start curvature:**
        - **G0, G1, G2, G3:** 始点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。
     - **End curvature:**
        - **G0, G1, G2, G3:** 終点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。
   -  **Trim:** オンにすると、ブリッジからはみ出たカーブはトリムされます。
4. コマンドダイアログでOKを押すか、右クリックで確定します。

## Bridge Vertex（ポイントをなめらかに繋げる）[#bridge-vertex]

連続性を考慮してふたつのポイントを繋げることができます。

1. 繋げたいふたつのポイントを選択します。
2. コマンドパレットで **Bridge** を実行します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **Tension 1:**
        - **Value 1:** 始点のテンション値(\<0,0\<)を指定します。
        - **Value 2:** 終点のテンション値(\<0,0\<)を指定します。
   - **Tension 2:**
        - **Value 1:** 
        - **Value 2:**
   - **Tension 3:**
        - **Value 1:** 
        - **Value 2:**
   - **Start curvature:**
        - **G0, G1, G2, G3:** 始点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。
     - **End curvature:**
        - **G0, G1, G2, G3:** 終点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。
4. コマンドダイアログでOKを押すか、右クリックで確定します。

## Bridge Edge（エッジをなめらかに繋げる）[#bridge-edge]

1. 繋げたいふたつのエッジを選択します。
2. コマンドパレットで **Bridge** を実行します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **Side** 
        - **Side 1**: 最初に選択したエッジの端点のどちら側にブリッジするかを指定します。
        - **Side 2**: 次に選択したエッジの端点のどちら側にブリッジするかを指定する。
   - **Tension 1:**
        - **Value 1:** 始点のテンション値(\<0,0\<)を指定します。
        - **Value 2:** 終点のテンション値(\<0,0\<)を指定します。
   - **Tension 2:**
        - **Value 1:** 
        - **Value 2:**
   - **Tension 3:**
        - **Value 1:** 
        - **Value 2:**
   - **Start curvature:**
        - **G0, G1, G2, G3:** 始点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。
     - **End curvature:**
        - **G0, G1, G2, G3:** 終点での[曲線の連続性](../cad-essentials/continuity-curve-and-surface)を指定します。 
4. コマンドダイアログでOKを押すか、右クリックで確定します。
