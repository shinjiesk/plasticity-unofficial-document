# Extend Curve（カーブを延長する）

<Reference shortcut="" command="Extend Curve" button="" />

カーブを延長できます。

## 基本的な使い方

1. 延長したいカーブ端の頂点を選択します。
2. コマンドパレットで **Extend** を実行します。
3. <kbd>D</kbd>を押し、カーソルを動かして、延長先の位置でクリックします。
4. コマンドダイアログでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Shape:**
         - **Natural** / **Liner** /  **Soft** /  **Reflective** / **Arc**
         - それぞれのオプションの違いは、動画を参照してください。
      - **Distance:** 距離を指定します。
5. コマンドダイアログでOKを押すか、右クリックで確定します。
