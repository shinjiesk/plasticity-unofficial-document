# Create Outline

<Reference shortcut="" command="Create Outline" button="" />

Projizieren Sie die Kontur der Solids oder Sheets auf die aktuelle Konstruktionsebene entlang der Normalenrichtung, um so Curves zu erzeugen. Während der **Project Outline**-Befehl eine projizierte Kurve in 2D erzeugt, erstellt dieser Befehl eine 3D Kurve. Die Curves werden genau an der ursprünglichen Grenze des Objekts positioniert.

## Grundlegende Nutzung

1. Legen Sie die gewünschte Konstruktionsebene für die Projektion fest.
2. Wählen Sie die Solids oder Sheets aus, die Sie projizieren möchten.
3. Verwenden Sie den **Create Outline** Befehl in der Befehlspalette.
4. Die projizierten Curves werden erstellt.
5. Optional können Sie die Curves mit dem Move Gizmo verschieben.
6. Klicken Sie mit der rechten Maustaste oder auf OK im Befehlsdialog, um zu bestätigen.
