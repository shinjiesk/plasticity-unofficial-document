# Project Curve Curve（ふたつのカーブを投影）[#project-curve-curve]

<Reference shortcut="I" command="Project Curve Curve" button="コマンドバー右"  buttonIcon='button-project'/>

ふたつのカーブを投影した交線のカーブを作成できます。

## 基本的な使い方

1. 投影したいカーブをふたつ選択します。
2. キーボードショートカット<kbd>I</kbd>またはボタンを押します。
3. 投影した交線のカーブが作成されます。
