# Trim

<Reference shortcut="T" command="Trim" button="Werkzeugleiste"  buttonIcon='button-trim'/>

Entfernt spezifizierte Segmente von planaren oder 3D Curves.

## Grundlegende Nutzung

1. Drücken Sie <kbd>T</kbd>.
2. Klicken Sie auf das Segment, das Sie trimmen möchten.
3. Das Segment wird entfernt.
