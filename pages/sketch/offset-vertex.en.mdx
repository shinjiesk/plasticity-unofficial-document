# Offset Vertex

<Reference shortcut="O" command="Offset Vertex" button="" />

Insert new vertices along the curve on both sides of a vertex.

## Basic Usage

1. Select a vertex.
2. Press <kbd>O</kbd>.
3. Move the mouse to determine the position of the new vertices.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
    - <Kst>Command Dialog</Kst>
        - **Distance:** Specify the distance.
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>D</kbd> **Distance:** Specify the distance with mouse movement.
5. Click to fix the position of the vertices.
5. Confirm with OK in the Command Dialog or right-click.
