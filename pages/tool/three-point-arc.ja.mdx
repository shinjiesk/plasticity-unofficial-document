# Three-Point Arc

<Reference shortcut='' command='Three-Point Arc' button='ツールバー'  buttonIcon='button-three-point-arc'/>

3点を指定して円弧を作成できます。

## 基本的な使い方

1. ツールバーのボタン <ButtonIcon type="button-three-point-arc" /> をクリックします。
2. 円周上の最初の位置でクリックします。
3. カーソルを移動して円周上の2点目でクリックします。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
4. カーソルを移動して円周上の3点目でクリックします。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd> : 寸法値に直接入力できるようになります。詳細は[こちら](/tool/sketching-essentials#specify-dimensions-numerically)を参照してください。
5. 円弧が作成されます。
