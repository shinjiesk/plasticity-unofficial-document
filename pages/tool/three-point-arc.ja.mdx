# Three-Point Arc（3点指定の円弧）

<Reference shortcut='' command='Three-Point Arc' button='ツールバー' />

円周上の3点で指定した円弧を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-three-point-arc" /> をクリックします。
2. 円周上の最初の位置でクリックします。
3. カーソルを移動して円周上の2点目でクリックします。
   - <kbd>X</kbd>, <kbd>Y</kbd>, <kbd>Z</kbd> **X, Y, Z-axis:** [座標軸方向に制限します](./sketching-tips#limit-to-the-coordinate-axis-direction)。
4. カーソルを移動して円周上の3点目でクリックします。
5. 円弧が作成されます。
