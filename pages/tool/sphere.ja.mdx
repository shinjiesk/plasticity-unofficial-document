# Sphere（球）

<Reference shortcut='' command='Sphere' button='ツールバー' buttonIcon='button-sphere'/>

中心と直径を指定して球を作成できます。

## 基本的な使い方

1. ツールバーのボタン <ButtonIcon type="button-sphere" /> をクリックします。
2. 球の中心でクリックします。
3. カーソルを動かして球の大きさを調整し、クリックして確定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Select target bodies :**
            - Booleanコマンドと同じ機能です。詳細は[**Boolean**](/solid/boolean)コマンドを参照してください。
      - **Radius :** 直径を指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd> : 寸法値に直接入力できるようになります。詳細は[こちら](/tool/sketching-essentials#specify-dimensions-numerically)を参照してください。
      - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd> :
         - Booleanコマンドと同じ機能です。詳細は[**Boolean**](/solid/boolean)コマンドを参照してください。
      - <kbd>D</kbd> **Radius :** 直径をマウス移動で指定します。
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis :** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
4. 黄色のドットを動かして形状を調整することもできます。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 球が作成されます。
