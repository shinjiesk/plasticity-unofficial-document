# Sphere（球）

<Reference shortcut='' command='Sphere' button='ツールバー' />

中心と直径を指定した球を作成できます

1. ツールバーのボタン <ButtonIcon type="button-sphere" /> をクリックします。
2. 球の中心でクリックします。
3. カーソルを動かして直径の位置でクリックします。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - これらのオプションはBooleanコマンドと同じ機能です。詳細は[**Boolean**](../solid/boolean)コマンドを参照してください。
   - **Radius** 直径を指定します。
   - <kbd>D</kbd> **Radius** 直径をマウス移動で指定します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 球が作成されます。
