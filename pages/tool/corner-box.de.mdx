# Corner Box

<Reference shortcut='Umschalt + B' command='Corner Box' button='Werkzeugleiste' buttonIcon='button-corner-box'/>

Sie können ein rechteckiges Feld erstellen, indem Sie die Diagonale seiner Basis und seine Höhe angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-corner-box" /> Knopf in der Werkzeugleiste.
2. Klicken Sie, um eine Ecke der Basis festzulegen.
3. Bewegen Sie den Cursor, um die Größe der Basis anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Details finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>C</kbd> **Center/Corner:** Wechselt zu Corner Box.
      - <kbd>S</kbd> **Square:** Beschränkt auf quadratische Basis.
4. Bewegen Sie den Cursor, um die Höhe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies**:
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - **Width:** Breite angeben.
      - **Length:** Länge angeben.
      - **Height:** Höhe angeben.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - <kbd>H</kbd> **Height:** Höhe durch Bewegen des Cursors anpassen.
5. Sie können auch die Form anpassen, indem Sie die gelben Punkte bewegen.
6. Klicken Sie im Befehlsdialog auf OK oder rechtsklick zur Bestätigung.
7. Das rechteckige Feld wird erstellt.
