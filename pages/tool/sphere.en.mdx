# Sphere

<Reference shortcut='' command='Sphere' button='Toolbar' />

Create a sphere by specifying its center and diameter. Boolean operations are also available.

1. Click the button <ButtonIcon type="button-sphere" /> on the toolbar.
2. Click at the point for the sphiere's center.
3. Move the cursor and click at the desired radius.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - These options have the same function as the Boolean command. See [**Boolean**](../solid/boolean) command for details.
   - **Radius:** Specify the radius.
   - <kbd>D</kbd> **Radius:** Move the cursor to specify the radius.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](./sketching-essentials#constrain-to-an-axis).
5. Confirm with OK in the Command Dialog or right-click.
6. A sphere is created.
