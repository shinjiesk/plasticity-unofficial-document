# Center Box

<Reference shortcut='' command='Center Box' button='Toolbar' />

Create a rectangular box by specifying the center, a corner point, and height. Boolean operations are also available.

1. Click the button <ButtonIcon type="button-center-box" /> on the toolbar.
2. Click at the center point of the base surface.
   - <kbd>C</kbd> **Center/Corner:** Switches to Corner Box.
   - <kbd>S</kbd> **Square:** Limits to a square.
3. Click at the endpoint of the perimeter.
4. Move the cursor and click at the desired height.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - These options have the same function as the Boolean command. See [**Boolean**](../solid/boolean) command for details.
   - **Width:** Specify the width.
   - **Length:** Specify the Length.
   - **Height:** Specify the height.
   - <kbd>H</kbd> **Height:** Moves the cursor to specify the height.
6. Confirm with OK in the Command Dialog or right-click.
7. A rectangular box is created.
