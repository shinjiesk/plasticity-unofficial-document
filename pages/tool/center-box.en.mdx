# Center Box

<Reference shortcut='' command='Center Box' button='Toolbar'  buttonIcon='button-center-box'/>

You can create a rectangular box by specifying the center of the base and the height.

## Basic Usage

1. Click the <ButtonIcon type="button-center-box" /> button on the toolbar.
2. Click the center point of the base.
3. Move the cursor to adjust the size of the base and click to confirm.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Tab</kbd>: Allows direct input of dimensions. For details, see [here](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>C</kbd> **Center/Corner:** Switches to Corner Box.
      - <kbd>S</kbd> **Square:** Restricts to a square shape.
4. Move the cursor to adjust the height and click to confirm.
   - <Kst>Command Dialog</Kst>
      - **Select target bodies**:
         - Same functionality as the Boolean command. For details, see the [**Boolean**](/solid/boolean) command.
      - **Width:** Specify the width.
      - **Length:** Specify the length.
      - **Height:** Specify the height.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>:
         - Same functionality as the Boolean command. For details, see the [**Boolean**](/solid/boolean) command.
      - <kbd>H</kbd> **Height:** Specify the height by moving the cursor.
5. You can also adjust the shape by moving the yellow dots.
6. Click OK in the command dialog or right-click to confirm.
7. The rectangular box is created.
