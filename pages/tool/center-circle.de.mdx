# Center Circle 

<Reference shortcut="" command="Center Circle" button="Werkzeugleiste" />

Erstellen Sie einen Kreis, indem Sie das Zentrum und den Durchmesser angeben. Optional können Sie den Winkel relativ zur Konstruktionsebene anpassen.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-center-circle" /> in der Werkzeugleiste.
2. Klicken Sie auf den Punkt für das Zentrum des Kreises.
3. Bewegen Sie den Cursor und klicken Sie auf einen Punkt am Umfang.
   - <kbd>Tab</kbd> [Den Durchmesser mit einem Zahlenwert angeben](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>K</kbd> **Knife:** [Verwenden Sie den Knife-Modus.](./sketching-essentials#use-knife-mode) 
   - <kbd>V</kbd> **Vertical/Horizontal:** Ändern Sie die Neigung in Bezug auf die Konstruktionsebene.
   - <kbd>X</kbd>/<kbd>Y</kbd> **X/Y-axis:** [Beschränken Sie sich auf die X- oder Y-Achse](./sketching-essentials#constrain-to-an-axis).
4. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
5. Ein Kreis wird erstellt.
