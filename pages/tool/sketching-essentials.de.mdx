# Skizzier-Grundlagen

## Exaktes Einrasten[#accurate-snap]

- Beim Verwenden von Curves für die Oberflächenmodellierung müssen Curves genau eingerastet werden.
- Wenn ein Vertex an eine Curve oder einen anderen Vertex einrastet, wechselt er von lila zu blau-grün.

![Tipps zum Skizzieren 1](/img/sketching-tips-1.png)

- Um genau zu rasten:
    - Halten Sie beim Verschieben eines Vertex <kbd>Strg</kbd> gedrückt, um das Einrasten zu aktivieren.
    - Beim Zeichnen in einer Skizze können Sie einrasten, ohne <kbd>Strg</kbd> zu halten, wenn **Object snapping** aktiviert ist.
    - Weitere Informationen finden Sie unter [Gitter und Fang](/plasticity-essentials/grid-and-snap).

## Einrastlinien als Referenz verwenden[#snap-reference-lines]

- Beim Zeichnen können Sie temporäre Referenzlinien erstellen, um das Einrasten zu unterstützen. Zeigen Sie mit dem Cursor auf ein Objekt und tippen Sie auf <kbd>Umschalt</kbd>. Einrastbare Referenzlinien werden angezeigt.

<VideoPlayer fileName="sketching-tips-2"/>

## Maße numerisch angeben[#specify-dimensions-numerically]

- Sie können die Maße und Winkel einer Skizze numerisch angeben: Drücken Sie die <kbd>Tab</kbd>-Taste, und ein numerisches Eingabefeld wird bei den Maßen angezeigt.

<VideoPlayer fileName="sketching-tips-3"/>

## Auf eine Achse beschränken[#constrain-to-an-axis]

- Drücken Sie die <kbd>X</kbd>/<kbd>Y</kbd> oder <kbd>Z</kbd>-Taste, um die Bewegung des Cursors auf eine einzelne Achse zu beschränken.

<VideoPlayer fileName="sketching-tips-4"/>

## Verwenden Sie den Knife-Modus[#use-knife-mode]

- Verwenden Sie den Knife-Modus, um eine oder mehrere Faces zu schneiden, während Sie eine Curve zeichnen. Sie müssen beim Zeichnen an Punkten auf den Ziel-Faces einrasten.

<VideoPlayer fileName="sketching-tips-5"/>
