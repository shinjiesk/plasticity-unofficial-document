# Three Point Rectangle

<Reference shortcut='' command='Three Point Rectangle' button='Werkzeugleiste' />

Erstellen Sie ein Rechteck, indem Sie drei Punkte angeben. Die Länge kann mit Mausaktionen oder mit einem numerischen Wert festgelegt werden.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-three-point-rectangle" /> in der Werkzeugleiste.
2. Klicken Sie auf den Startpunkt einer Seite des Rechtecks.
   - <kbd>Tab</kbd> [Länge numerisch angeben](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>K</kbd> **Knife:** [Messermodus verwenden.](./sketching-essentials#use-knife-mode) 
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Einschränken auf die X-, Y- oder Z-Achse](./sketching-essentials#constrain-to-an-axis).
3. Klicken Sie auf den Endpunkt derselben Seite des Rechtecks.
4. Klicken Sie auf den dritten Punkt des Rechtecks.
5. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
6. Ein Rechteck wird erstellt.
