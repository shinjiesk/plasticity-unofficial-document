# Spline Curve

<Reference shortcut="" command="Spline Curve" button="Toolbar" />

Create a Curve by placing points which the Curve will pass through.

1. Click the button <ButtonIcon type="button-spline-curve" /> on the toolbar.
2. Click at the start point of the spline curve.
3. Click the points through which the  Curve will pass. Repeat the clicks to create a spline curve.
   - <kbd>K</kbd> **Knife:** [Use Knife Mode.](./sketching-essentials#use-knife-mode) 
   - <kbd>Tab</kbd> **Control / Through-point:** Switches to Control Point Curve.
   - <kbd>Ctrl + Z</kbd> **Undo:** Cancel the last click.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](./sketching-essentials#constrain-to-an-axis).
4. Confirm with OK in the Command Dialog or right-click.
5. A Curve is created.

<Callout type="info" emoji="ðŸ’¡">
**Differences Between Two Curves**
- Spline Curve: Click the points that the  Curve will pass through to create the  Curve.
- Control Point Curve: Click the Control Points of the  Curve to create the  Curve.
</Callout>
