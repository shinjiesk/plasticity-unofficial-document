# Cylinder

<Reference shortcut='Umschalt + C' command='Cylinder' button='Werkzeugleiste' />

Erstellen Sie einen Zylinder, indem Sie das Zentrum eines Kreises, den Durchmesser und die Höhe angeben. Es sind auch boolesche Operationen verfügbar.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-cylinder" /> in der Werkzeugleiste.
2. Klicken Sie auf den Punkt für das Zentrum des Kreises auf der Basisebene.
3. Bewegen Sie den Cursor und klicken Sie auf einen Punkt am Umfang.
   - <kbd>V</kbd> **Vertical/ Horizontal:** Ändert die Neigung in Bezug auf die Konstruktionsebene.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Auf die X-, Y- oder Z-Achse beschränken](./sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor und klicken Sie auf die gewünschte Höhe.
5. Legen Sie die Optionen im Command Dialog und den Tastaturkürzeln fest.
   - **Select target bodies**, <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - Diese Optionen haben die gleiche Funktion wie der Befehl Boolean. Weitere Einzelheiten finden Sie im Befehl [**Boolean**](../solid/boolean).
   - **Radius:** Geben Sie den Radius an.
   - **Height:** Geben Sie die Höhe an.
   - <kbd>F</kbd> **Radius:** Bewegen Sie den Cursor, um den Radius anzugeben.
   - <kbd>D</kbd> **Height:** Bewegen Sie den Cursor, um die Höhe anzugeben.
6. Bestätigen Sie mit OK im Command Dialog oder mit Rechtsklick.
7. Ein Zylinder wird erstellt.
