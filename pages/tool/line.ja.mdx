# Line（線分）

<Reference shortcut="Shift + A" command="Line" button="ツールバー" />

連続した線分を作成できます。長さと角度は数値でも指定できます。

1. ツールバーのボタン <ButtonIcon type="button-line" /> をクリックします。
2. 線分の最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、一連の線分を作成します。
    - <kbd>Tab</kbd> [長さと角度を数値で指定します](./sketching-essentials#specify-dimensions-numerically)。
   - <kbd>K</kbd> **Knife:** [ナイフモードを使う](./sketching-essentials#use-knife-mode)を参照してください。
    - <kbd>Ctrl + Z</kbd> **Undo:** 直前のクリックを取り消します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
5. 線分が作成されます。
