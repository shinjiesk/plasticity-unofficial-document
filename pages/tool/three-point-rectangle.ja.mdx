# Three Point Rectangle（3点指定の四角形）

<Reference shortcut='' command='Three Point Rectangle' button='ツールバー'  buttonIcon='button-three-point-rectangle'/>

3点を指定して長方形を作成できます。

## 基本的な使い方

1. ツールバーのボタン <ButtonIcon type="button-three-point-rectangle" /> をクリックします。
2. 四角形の角でクリックします。
3. 四角形のもう1点をクリックします。
4. カーソルを動かして大きさを調整し、クリックして確定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd> : 寸法値に直接入力できるようになります。詳細は[こちら](/tool/sketching-essentials#specify-dimensions-numerically)を参照してください。
      - <kbd>K</kbd> **Knife:** [ナイフモードを使う](/tool/sketching-essentials#use-knife-mode)を参照してください。
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
3. 右クリックで確定します。
5. 四角形が作成されます。
