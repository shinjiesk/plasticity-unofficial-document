# Revolve

<Reference shortcut="" command="Revolve" button="Right of Command Bar" buttonIcon="button-revolve" />

Revolving Regions generates Solids, while revolving Curves produces either Solids or Sheets. Options include Normal, Binormal, and Tangent.

## Basic Usage

1. Select the Region or Curve to revolve.
2. Click the button <ButtonIcon type="button-revolve" /> on the Right of Command Bar.
3. Specify the revolution axis. 
   - Revolve along the axis defined by the first click as the start point and the second click as the endpoint.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Shift + T</kbd> **Thickness:**
      - <kbd>A</kbd> **Angle:** Move the mouse to specify the rotation angle.
5. Confirm with OK in the Command Dialog or right-click.
