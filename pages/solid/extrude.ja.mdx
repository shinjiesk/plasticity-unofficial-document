# Extrude（押し出す）

<Reference shortcut="E" command="Extrude" button="コマンドバー右" buttonIcon="button-extrude"/>

フェイスまたはリージョンを押し出したときはソリッドが作成されます。カーブのときはシートが作成されます。

## 基本的な使い方

1. <kbd>E</kbd>を押すか、ボタン <ButtonIcon type="button-extrude" /> をクリックします。
2. 押し出すフェイス、リージョンまたはカーブを選択します。
3. 黄色い丸を動かして距離を指定します。
4. コマンドダイアログまたはキーボードショートカットでオプションを設定します。 
   - <Kst>コマンドダイアログ</Kst>
      - **Select target bodies:**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>,  <kbd>B</kbd>,  <kbd>T</kbd>
         - これらのブーリアンオプションは、Booleanコマンドと同じ機能です。詳細は[**Boolean**](boolean)コマンドを参照してください。
      - **Distance:** 距離と反対側の距離を指定します
      - **Degrees:** 押し出しの角度を指定します。
      - **Thickness:** 壁の厚さを指定します。
      - **Keep tools:** <kbd>T</kbd>と同じです。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>F</kbd> **Freestyle** クリックして始点と終点を指定し、移動の方向と距離を指定します。
      - <kbd>V</kbd> **Pivot** クリックしてギズモの位置を指定します。
      - <kbd>D</kbd> **Distance 1** カーソルを動かして距離を指定します。
      - <kbd>A</kbd> **Angle 1** カーソルを動かして押し出しの角度を指定します。
      - <kbd>Shift + T</kbd> **Thickness** カーソルを動かして壁の厚さを指定します。
      - <kbd>Tab</kbd> **Lock distances** 両側の距離を同じにして押し出します。
5. コマンドダイアログのOKをクリックするか、右クリックします。
