# Imprint

このコマンドは、選択されていたオブジェクトに応じて以下のコマンドを自動的に呼び出します。
- カーブ、ソリッド、シートが選択された場合、または何も選択されていない場合: 以下に説明する[**"Imprint Curve Body"**](#imprint-curve-body)を参照してください。
- 2つのソリッドまたは/およびシートが選択された場合: 以下に説明する[**"Imprint Body Body"**](#imprint-body-body)を参照してください。



## Imprint Curve Body（カーブをボディに投影してエッジを作成する）[#imprint-curve-body]

カーブをソリッドまたはシートに投影してエッジを作成できます。

### カーブが選択された場合:

1. <kbd>Shift-I</kbd>を押します。
2. カーブを投影するソリッドまたはシートを選択します。
3. ステップ4に進みます。

### ソリッドまたはシートが選択された場合:

1. <kbd>Shift-I</kbd>を押します。
2. ソリッドまたはシートに投影するカーブを選択します。
3. ステップ4に進みます。

### 何も選択されていない場合:

1. <kbd>Shift-I</kbd>を押します。
2. ソリッドまたはシートに投影するカーブを選択します。
3. カーブを投影するソリッドまたはシートを選択します。
4. コマンドダイアログでオプションを設定します。
    - **Method:**
        - **Normal:** 標準的な方法で投影します。
        - **Vector:** **Direction**で指定した方向に投影します。**Bidirectional**と**X-Ray**が有効になります。
    - **Direction:**
        - **Value 1:** 投影方向のベクトルのX成分を指定します。
        - **Value 2:** 投影方向のベクトルのY成分を指定します。
        - **Value 3:** 投影方向のベクトルのZ成分を指定します。
    - **Bidirectional:** 投影を両方向に行います。
    - **X-Ray:** 
        - **Hide occlusion:** オンにすると、オブジェクトの背後の面への投影を行いません。
    - **Completion:** 投影したエッジの延長を指定します。
        - **None:** 投影したエッジの延長をしません。
        - **Edge:** 投影したエッジが他のエッジにあたるまで延長します。
        - **Boundary:** 投影したエッジが他のエッジにあたるまで延長し、そのエッジをつなぎます。
5. コマンドダイアログのOKまたは右クリックで確定します。

## Imprint Body Body（ボディをボディに投影してエッジを作成する）[#imprint-body-body]

2つのソリッドまたは/およびシートの交点にエッジを作成できます。

1. 交点のターゲットとして使用するソリッドまたはシートを選択します。
2. 交点のツールとして使用するソリッドまたはシートを選択します。
3. コマンドダイアログでオプションを設定します。
    - **Imprint tool:** **Imprint**をオンにすると、**Complete tool** オプションが有効になります。
    - **Complete target:**
        - **None:** ターゲットボディ上の投影したエッジの延長をしません。
        - **Edge:** ターゲットボディ上の投影したエッジが他のエッジにあたるまで延長します。
    - **Complete tool:**
        - **None:** ツールボディ上の投影したエッジの延長をしません。
        - **Edge:** ツールトボディ上の投影したエッジが他のエッジにあたるまで延長します。  
4. コマンドダイアログのOKまたは右クリックで確定します。
