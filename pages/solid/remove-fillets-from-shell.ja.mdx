# Remove Fillets From Shell（フィレットを削除する）

フィレットの最大半径や凸凹性を指定して、ソリッドやシートからフィレットを削除することができます。

1. フィレットを削除したいソリッドやシートを選択します。
2. コマンドパレットで **Remove Fillets From Shell** コマンドを実行します。
3. コマンドダイアログでオプションを設定します。
    - **Max Radius:** 削除するフィレットの最大半径を指定します。
    - **Convexity:** 削除するフィレットの凸凹性を指定します。
        - **Any:** すべての凸凹のフィレットを削除します。
        - **Convex:** 凸のフィレットを削除します。
        - **Concave:** 凹のフィレットを削除します。
4. コマンドダイアログで OK をクリックするか、右クリックして確定します。
