# Unjoin Faces

<Reference shortcut="Alt + J" command="Unjoin Faces" button="" />

Ausgewählte Faces von Solid oder Sheet trennen.

## Grundlegende Nutzung

1. Wählen Sie die Faces aus, die Sie trennen möchten.
2. Drücken Sie <kbd>Alt + J</kbd>.
3. Die Faces werden getrennt.
