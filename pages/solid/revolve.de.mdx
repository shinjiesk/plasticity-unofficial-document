# Revolve

<Reference shortcut="" command="Revolve" button="Rechts von Command Bar" />

Das Drehen von Regionen erzeugt Solids, während das Drehen von Curves entweder Solids oder Sheets produziert. Optionen beinhalten Normal, Binormal und Tangent.

1. Wählen Sie die Region oder Curve zum Drehen aus.
2. Klicken Sie auf den Knopf.
3. Legen Sie die Drehachse fest. 
   - Entlang der Achse drehen, die durch den ersten Klick als Startpunkt und den zweiten Klick als Endpunkt definiert wird.
   - <kbd>Umschalt + T</kbd> **Thickness:**
   - <kbd>A</kbd> **Angle:** Bewegen Sie die Maus, um den Drehwinkel zu bestimmen.
5. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
