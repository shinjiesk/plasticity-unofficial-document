# Pipe（パイプを作成）

<Reference shortcut="P" command="Pipe" button="コマンドバー右" buttonIcon="button-pipe"/>

カーブやエッジに沿ったパイプを作成できます。

## 基本的な使い方

1. パイプにしたいカーブまたはエッジを選択します。複数のカーブまたはエッジを選択したときは、複数のパイプが作成されます。
2. <kbd>P</kbd> を押します。
3. 黄色い点をドラッグして断面サイズを、黄色い四角をドラッグしてパイプの厚みを指定します。白い丸をドラッグして回転角度を指定します。
4. コマンドダイアログやキーボードショートカットでオプションを設定します。
     - <Kst>コマンドダイアログ</Kst>
          - **Select target bodies:**
               - Booleanコマンドと同じ機能です。詳細は[**Boolean**](boolean)コマンドを参照してください。
          - **Select custom profile:** カスタムプロファイルにするリージョン、カーブ、フェースを選択します。
          - **Vertex count:** プロファイルの多角形の頂点数を指定します。0を指定すると円形になります。
          - **Section size:** 断面サイズを指定します。
          - **Thickness:** パイプの厚みを指定します。
          - **Angle:** 回転角度を指定します。
     - <Kst>キーボードショートカット</Kst>
          - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>,  <kbd>B</kbd>,  <kbd>T</kbd>
               - Booleanコマンドと同じ機能です。詳細は[**Boolean**](boolean)コマンドを参照してください。
          - <kbd>C</kbd> **Custom profile** カスタムプロファイルにするリージョン、カーブ、フェースを選択します。
          - <kbd>D</kbd> **Section size** 断面サイズをカーソルを動かして指定します。
          - <kbd>A</kbd> **Angle** 回転角度をカーソルを動かして指定します。
          - <kbd>Shift + T</kbd> **Thickness** パイプの厚みをカーソルを動かして指定します。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。

<Callout emoji="❗">
**注意**<br />
Pipeコマンドは、正確な寸法のパイプを作るためのものではありません。
見た目がクールなパイプやチューブを素早く作成するためのものです。
ポリゴンとして角度をつけるなど、多くのことを行うので、配管などには寸法的に正確ではありません。
正確なパイプを作るには、[Sweep](sweep) コマンドを使ってください。
</Callout>
