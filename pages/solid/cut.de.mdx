# Cut

<Reference shortcut="C" command="Cut" button="Links von der Befehlsleiste"  buttonIcon="button-cut" />

Schneiden Sie das Ziel-Solid oder Sheet mit einer Cutter-Curve oder Face. Wenn eine Curve der Cutter ist, wird der Schnitt basierend auf einer von dieser Curve erzeugten Oberfläche durchgeführt.

## Grundlegende Nutzung

1. Drücken Sie <kbd>C</kbd>, um den Befehl auszuführen.
2. Spezifizieren Sie das Ziel und den Cutter.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies:** 
         - Wählen Sie das Solid oder Sheet aus, das geschnitten werden soll.
         - Um das Ziel zu ändern, klicken Sie auf **Select target bodies** und wenn es hervorgehoben ist, wählen Sie das Ziel.
      - **Select cutters:** 
         - Wählen Sie die Curve oder Face aus, die als Cutter verwendet werden soll.
         - Um die Cutter zu ändern, klicken Sie auf **Select cutters** und wenn es hervorgehoben ist, wählen Sie die Cutter.
   - <kbd>C</kbd> **Cut Curves:** Wechseln Sie zum **Cut Curve** Befehl. Siehe [**Cut Curve**](/sketch/cut-curve)
   - <kbd>S</kbd> **Screen space:** Legt die Schnittrichtung fest, sodass sie mit der sichtbaren Richtung übereinstimmt.
3. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
