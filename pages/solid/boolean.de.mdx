# Boolean

<Reference shortcut="Q" command="Boolean" button="Links von der Befehlsleiste"  buttonIcon="button-boolean"/>

Kombinieren, subtrahieren oder schneiden Sie mehrere Objekte in beliebiger Kombination von Solids und Sheets.

## Grundlegende Verwendung

## Grundlegende Nutzung

1. Drücken Sie <kbd>Q</kbd>, um den Befehl auszuführen.
2. Wählen Sie das Ziel und das Werkzeug aus.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies:** 
         - Wenn dieser Button hervorgehoben ist, können Sie den Solid oder Sheet auswählen, der als Boolean-Ziel verwendet werden soll.
         - Um das Ziel zu ändern, klicken Sie erneut auf diesen Button und wählen den Solid oder Sheet.
      - **Select tool bodies:** 
         - Wenn dieser Button hervorgehoben ist, können Sie den Solid oder Sheet auswählen, der als Boolean-Werkzeug verwendet werden soll.
         - Um das Werkzeug zu ändern, klicken Sie erneut auf diesen Button und wählen den Solid oder Sheet.
3. Legen Sie die Optionen im Befehlsdialog und die Tastaturkürzel fest.
   - <Kst>Befehlsdialog</Kst>
      - **Keep Tools:** Führt Boolean-Operationen durch und behält die Werkzeugobjekte bei.
      - **Opetation:**
         - **Union:** Verbindet das Boolean-Ziel und das Werkzeug miteinander.
         - **Diff(erence):** Entfernt das Werkzeug vom Boolean-Ziel.
         - **Intersect:** Erstellt einen Solid an der Schnittstelle des Boolean-Ziels und des Werkzeugs.
         - **Slice:** Erstellt einen Solid an den Schnittstellen mit dem Boolean-Ziel und bewahrt die verbleibenden Solids.
         - Bei Unterschieden in den Ergebnissen aufgrund der Operation, bitte beziehen Sie sich auf den [Operation Outcome Differences](#operation-outcome-differences) Abschnitt.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Shift + Q</kbd> **Slice:** Wie oben beschrieben.
      - <kbd>Q</kbd> **Union:** Wie oben beschrieben.
      - <kbd>W</kbd> **Difference:** Wie oben beschrieben.
      - <kbd>Shift + E</kbd> **Intersect:** Wie oben beschrieben.
      - <kbd>T</kbd> **Keep Tool:** Wie oben beschrieben.
      - <kbd>G</kbd> **Move:** Bewegen Sie das Werkzeug auf die gleiche Weise wie den **Move** Befehl.
      - <kbd>R</kbd> **Rotate:** Drehen Sie das Werkzeug auf die gleiche Weise wie den **Rotate** Befehl.
      - <kbd>S</kbd> **Scale:** Skalieren Sie das Werkzeug auf die gleiche Weise wie den **Scale** Befehl.
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Auf X-, Y- oder Z-Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
      - <kbd>Shift + X</kbd>/<kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z plane:** Auf die X-, Y- oder Z-Ebene beschränken.
4. Bestätigen Sie mit OK im Befehlsdialog oder Rechtsklick.
