# Extend Edge

<Reference shortcut="" command="Extend Edge" button="" />

Extend the selected Edge until it intersects with another Edge.

## Basic Usage

1. Select Edges you want to extend.
2. Use the **Extend Edge** command in the Command Palette.
3. The Edge is extended.
