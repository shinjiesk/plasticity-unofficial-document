# Fillet Shell

<Reference shortcut="B" command="Fillet Shell" button="" />

ソリッドまたはシートをフィレットまたは面取りすることができます。

## 基本的な使い方

1. フィレットまたは面取りしたいエッジを選択します。
2. エッジが選択されると、デフォルトのコマンドであるこのコマンドが自動的に実行されます。
3. または、ショートカットキー<kbd>B</kbd>を押して実行します。
4. 黄色い点を動かして、フィレットまたは面取りの距離を指定します。
    - 正の方向に動かす: フィレット
    - 負の方向に動かす: 面取りに
    <VideoPlayer fileName="fillet-solid-0"/>
4. さらにエッジを追加で選択するには、<kbd>Ctrl</kbd>キーを押しながらクリックします。選択を解除するには、<kbd>Ctrl</kbd>キーを押しながら再度クリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。下記の[**フィレットオプション**](#fillet-options) と [**面取りオプション**](#chamfer-options) を参照してください。
6. コマンドダイアログのOKボタンをクリックするか、右クリックして確定します。

## フィレットオプション [#fillet-options]

### コマンドダイアログ

- **Distance:** 
    - エッジをフィレットするには、正の値を入力します。エッジを面取りするには、負の値を入力します。
<Callout type="info" emoji="💡">
    **Tip**
    フィレットの半径をコピーするには、コピーしたいフィレットをクリックします。ただし、2つの**Distance**が同じで、形状が**Conic**の場合に限ります。
<VideoPlayer fileName="fillet-solid-copyfillet"/>
</Callout>

**Shape:**
    - **Conic:** （円錐曲線）一般的な円形のフィレットよりも自然でなめらかな遷移が得られる特性があります。エアロダイナミクスや見た目が重要な場合に特に有用です。
    - **Chordal:** （弦）フィレットの弦の長さが一定に保たれる特性があります。角度が異なる角にフィレットを適用した場合でも、フィレット部分の長さが一定となります。これは見た目の一貫性を保ちたい場合に有用です。
    - **G2:** （曲率連続）フィレットの両端の面と滑らかに連続するような曲線を生成します。この設定は、形状の遷移が非常に滑らかで自然な形状を求める高度なデザインや複雑な曲面処理に適しています。[サーフェスの連続性](/cad-essentials/continuity-curve-and-surface)を参照してください。
    <VideoPlayer fileName="fillet-solid"/>

**Y-Blend:** 
    - Yブレンド "は、3つのサーフェスが共通の点で合流し、"Y "形状を形成するときに発生するブレンドまたはフィレットの特別な種類です。**Attempt**を有効にすると、Yブレンドでより良いトポロジーを作成し、望ましくないフィレットの結果を避けることができます。
    <VideoPlayer fileName="fillet-solid-yblend"/>

**Selection:** 
    - Disable **Add tangent Edges** 選択したエッジのみにフィレットを制限します。
    <VideoPlayer fileName="fillet-sold-selection"/>

### ショートカット

<kbd>A</kbd> **: Chamfer angle**
    - カーソルを動かして、フィレットの角度を指定します。
<Callout type="info" emoji="💡">
    距離のふたつの値を変更することでも、角度を変えることができます。
    <VideoPlayer fileName="fillet-solid-two-value"/>
</Callout>

<kbd>D</kbd> **: Fillet distance** 
    - カーソルを動かして、フィレットの距離を指定します。

<kbd>C</kbd> **: Chamfer distance** 
    - カーソルを動かして、面取りの距離を指定します。

<kbd>V</kbd> **: Add variable point**
    - フィレットに可変点を追加します。
    <VideoPlayer fileName="fillet-solid-variable"/>

<kbd>L</kbd> **: Add limit (stop) point** <br />
    - フィレットする範囲を指定します。[**オブジェクトスナップ**](/plasticity-essentials/grid-and-snap#object-snap)が有効になっていることを確認し、フィレットの中心線をクリックしてリミットポイントを追加します。方向を逆にするには、リミットポイントをもう一度クリックします。リミットポイントは2つまで設定できます。
    <VideoPlayer fileName="fillet-solid-limit"/>

### 面取りオプション [#chamfer-options]
- **Shape:**
    - **Offset:** デフォルトの面取り。
    - **Apex:** 面取りの境界線のずれを減らします。
- そのほかはフィレットと同じです。
