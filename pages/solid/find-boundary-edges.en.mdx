# Find Boundary Edges

<Reference shortcut="" command="Find Boundary Edges" button="Right of Command Bar"  buttonIcon="button-find-boundary-edges"/>


Select the boundary Edges of an open area on a selected Sheet.

## Basic Usage

1. Select a Sheet you want to find boundary Edges.
2. Click the button <ButtonIcon type="button-find-boundary-edges" /> on the Right of Command Bar.
3. Edges of the Sheet opening are selected.
