# Rundungen vom Gehäuse entfernen

<Reference shortcut="" command="Remove Fillets From Shell" button="" />


Entfernen von Rundungen aus Solid oder Sheet mit möglichen Einschränkungen für den maximalen Radius und die Konvexität.

## Grundlegende Nutzung

1. Wählen Sie das Solid oder Sheet aus, von dem Sie die Rundungen entfernen möchten.
2. Verwenden Sie den Befehl **Remove Fillets From Shell** in der Befehlspalette.
3. Legen Sie die Optionen im Befehlsdialog fest.
    - <Kst>Befehlsdialog</Kst>
        - **Max Radius:** Geben Sie den maximalen Radius der zu entfernenden Rundungen an.
        - **Convexity:** Spezifizieren Sie die Konvexität der zu entfernenden Rundungen.
            - **Any:** Entfernen Sie Rundungen jeder Konvexität.
            - **Convex:** Entfernen Sie konvexe Rundungen.
            - **Concave:** Entfernen Sie konkave Rundungen.
4. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
