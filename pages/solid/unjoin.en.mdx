# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="" />

Unified command for **Unjoin Curves**, **Unjoin Faces**, **Unjoin Shells**. The command is run based on the selected object.
    - When Curves are selected: see [**Unjoin Curves**](/sketch/unjoin-curve) in the sketch section.
    - When Faces are selected: see [**Unjoin Faces**](unjoin-faces).
    - When Solids or Sheets are selected: see [**Unjoin Shells**](unjoin-shells).
