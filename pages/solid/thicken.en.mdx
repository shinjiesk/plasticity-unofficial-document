# Thicken

<Reference shortcut="" command="Thicken" button="Right of Command Bar" buttonIcon="button-thicken" />

Add thickness to a selected Sheet. Thickness for each side can be specified.<br />
When Faces are selected: see [**Thicken Face**](thicken-face).

## Basic Usage

1. Select the Sheet you want to thicken.
2. Click the button <ButtonIcon type="button-thicken" /> on the Right of Command Bar.
3. Set the options in the Command Dialog and Keyboard Shortcuts.
    - <Kst>Command Dialog</Kst>
        - **Thickness:** Specify the thickness of each of both sides.
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>Tab</kbd> **Lock Distances** Equalize the thickness on both sides.
        - <kbd>D</kbd> **Thickness** Specify the thickness by moving the cursor.
4. Confirm with OK in the Command Dialog or right-click.
