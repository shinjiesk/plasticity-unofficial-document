# Delete Redundant Topology

<Reference shortcut="" command="Delete Redundant Topology" button="" />


This command automatically calls the following command depending on the selected object.
    - When Curves are selected: see [**Delete Redundant Topology (Curve)**](/sketch/delete-redundant-topology#delete-redundant-topology-curve) in the Sketch Commands section.
    - When Solids or Sheets are selected: see **Delete Redundant Topology (Solid)** described below.

## Delete Redundant Topology (Solid)[#delete-redundant-Topology-solid]

You can delete redundant topology from Solids or Sheets.

1. Select the Solids or Sheets to delete redundant topology from.
2. Use the **Delete Redundant Topology** command in the Command Palette.
3. Redundant topologies are deleted.
