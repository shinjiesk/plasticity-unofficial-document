# Loft

<Reference shortcut="L" command="Loft" button="コマンドバー右"   buttonIcon='button-loft' />

## カーブとエッジをロフトする

2つ以上のプロファイルカーブからサーフェスを作成できます。

### カーブをロフトする

カーブのロフトでは、**Add Limit Point**オプションを指定できます。

1. ロフトしたい2つのカーブを選択します。
2. <kbd>L</kbd>を押します。
3. キーボードショートカットでオプションを設定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>L</kbd> **Add Limit Point:** ロフトの範囲を指定します。
            - [**Object Snaps**](/plasticity-essentials/grid-and-snap#object-snap)が有効になっていることを確認してください。
            - エッジをクリックしてリミットポイントを追加します。
            - 方向を反転するには、リミットポイントを再度クリックします。
            - 各カーブには最大2つのリミットポイントを設定できます。
4. コマンドダイアログでOKをクリックするか、右クリックします。

<VideoPlayer fileName="loft-curve-and-curve"/>

### エッジをロフトする

エッジのロフトでは、**Continuity**、**Tension**、**Add Limit Point**オプションを指定できます。

1. ロフトしたい2つのエッジを選択します。
2. <kbd>L</kbd>を押します。
3. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Continuity**: **G0, G1, G2**
            - サーフェイスの連続性を指定します。詳しくは[continuity of the Surface](/cad-essentials/continuity-curve-and-surface)を参照してください。
            - サーフェイス上の黒い丸をクリックして、サーフェイスの連続性を個別に指定できます。
        - **Tension**: 各エッジのテンションを指定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>Tab</kbd> **Cycle continuity:** 押すごとに連続性が切り替わります。
        - <kbd>D</kbd> **Tension:** カーソルを動かすことで、両方のエッジのテンションを一度に指定できます。
        - <kbd>L</kbd> **Add Limit Point:** ロフトの範囲を指定します。**2つのカーブをロフトする**と同じです。
4. コマンドダイアログでOKをクリックするか、右クリックします。

<VideoPlayer fileName="loft-edge-and-edge"/>

### カーブとエッジをロフトする

カーブとエッジは任意の組み合わせでロフトできます。手順と機能は上述の**カーブをロフトする**と**エッジをロフトする**と同じです。

## 複数のカーブをロフトする

一度に複数のカーブをロフトできます。複数選択するには<kbd>Shift</kbd> + クリック、選択を解除するには<kbd>Ctrl</kbd> + クリックします。手順と機能は**カーブをロフトする**と同じです。始点と終点にエッジを指定することもできます。

<VideoPlayer fileName="loft-multiple-curves"/>

## ガイド付きでロフトする

ガイドでロフトの形状を制御できます。ガイドにはカーブとエッジの任意の組み合わせを指定できます。ロフトしたサーフェイスはトリムできます。

1. ロフトしたい2つのカーブを選択します。
2. <kbd>L</kbd>を押します。
3. ガイドになるカーブまたはエッジを選択します。複数選択するには<kbd>Shift</kbd> + クリック、選択を解除するには<kbd>Ctrl</kbd> + クリックします。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Trim overlap:**
            - **Guides:** ガイド側のロフトサーフェイスをトリムします。
            - **Profiles:** プロファイル側のロフトサーフェイスをトリムします。
    - そのほかのオプションは、上述の**カーブをロフトする**と**エッジをロフトする**と同じです。
5. コマンドダイアログでOKをクリックするか、右クリックします。

<VideoPlayer fileName="loft-curve-with-guide"/>

## 連続ロフト

ひとつだけエッジを選択してロフトすると、隣接する2つのエッジをガイドにしてロフトできます。この機能を連続して行うと、ふたつのシートを容易に繋げることができます。

<VideoPlayer fileName="loft-continuous"/>

## ループ状にロフトする

3つ以上のカーブをロフトしてループ状のシートを作成できます。

1. ロフトしたい3つ以上のカーブを選択します。
2. <kbd>L</kbd>を押します。
3. コマンドダイアログで**Loop**をクリックします。
4. キーボードショートカットでオプションを設定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>L</kbd> **Add Limit Point:** ロフトの範囲を指定します。**2つのカーブをロフトする**と同じです。
4. コマンドダイアログでOKをクリックするか、右クリックします。

<VideoPlayer fileName="loft-loop"/>

## 閉じたカーブをロフトする

閉じたカーブをロフトして、閉じたシートを作成できます。

1. ロフトしたい2つ以上の閉じたカーブを選択します。
2. <kbd>L</kbd>を押します。
3. ロフトの形状を制御するには、ガイドになるカーブを選択します。
4. コマンドダイアログでOKをクリックするか、右クリックします。

## リージョンをロフトする

リージョンをロフトして、ソリッドを作成できます。

1. ロフトしたい2つ以上のリージョンを選択します。
2. <kbd>L</kbd>を押します。
3. ロフトの形状を制御するには、ガイドになるカーブを選択します。
4. コマンドダイアログでOKをクリックするか、右クリックします。

## フェースをロフトする

ふたつのフェースをロフトして、ソリッドを作成できます。

1. ロフトしたい2つのフェースを選択します。
2. <kbd>L</kbd>を押します。
3. ロフトの形状を制御するには、ガイドになるカーブを選択します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Continuity**: **G0, G1, G2** 
            - サーフェイスの連続性を指定します。詳しくは[continuity of the Surface](/cad-essentials/continuity-curve-and-surface)を参照してください。
            - サーフェイス上の黒い丸をクリックして、サーフェイスの連続性を個別に指定できます。
        - **Tension**: 各エッジのテンションを指定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>Tab</kbd> **Cycle continuity:** 押すごとに連続性が切り替わります。
4. コマンドダイアログでOKをクリックするか、右クリックします。

## 頂点をロフトする

ふたつの頂点をロフトすると、ふたつのカーブをブリッジできます。機能は[**Bridge Vertex**](/sketch/bridge#bridge-vertex)と同じです。

## トラブルシューティングのヒント [#troubleshooting-tips]

<Callout type="info" emoji="💡">
**ロフトが成功しない理由として考えられること**
1. カーブが複雑すぎる。 カーブを作り直すことで解決することが多いです。
2. ガイドカーブがプロファイルに接触していない。 各接続部に交差を示す青緑色の点があることを確認してください。 正確に接続するには、[**オブジェクトスナップ**](/plasticity-essentials/grid-and-snap#object-snap)を参照してください。
3. 複雑な形状を一度にLoftしようとする。 複数のロフトを使用した、より単純なサブセクション形状を使用してモデルを設計します。例えば、**Join** を使用してサーフェスを接続したり、追加の **Loft** を使用してブリッジカーブを使用します。
4. ガイドカーブの数が多すぎる、または足りない。ここは練習すれば上達します。サーフェスを無理に多くのプロファイルやガイドに通そうとすると、ロフトが困難になります。プロファイルとガイドの間隔が近すぎる場合も同じ問題があります。
5. エッジの代わりにカーブを選択する。エッジとカーブでは選択できるオプションが異なります。
6. カーブが正しい順序で選択されていない。プロファイルとガイドは、サーフェスが流れる順序で選択する必要があります。
7. ロフトが失敗しても、失敗ではない場合がある。 一般的に、ガイドを選択するとロフトが伸びていくのがわかります。しかし、次のガイドを選択するときに、サーフェスが現れたり消えたりすることがあります。 ロフトが失敗したと感じますが、次のガイドに進むと、すべてのサーフェスが突然現れることがあります。 まるでPariSolidがパズルを解くために、最初にすべての入力を必要としているかのようです。
8. プロファイルとガイドの選択順序を逆にする。 通常はプロファイルを選択し、次にガイドを選択します。 ロフトが失敗したら、「ガイド」を選択してから「プロファイル」を選択してみてください。 これは逆に見えてもうまくいくことがあります。
</Callout>
