# Offset Edge（エッジをオフセットする）

<Reference shortcut="O" command="Offset Edge" button="コマンドバー右"  buttonIcon="button-offset-curve"/>

エッジをオフセットして、新しいエッジを作成することができます。

## 基本的な使い方

1. オフセットしたいエッジを選択します。
2. <kbd>O</kbd>を押します。
3. マウスを左から右に動かすことで、方向と距離を指定します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Distance:** 距離を指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>V</kbd> **Gap fill:**
         - **Round:** 出っ張った角を丸めます。
         - **Linear:** 出っ張った角を直線でつなぎます。
         - **Natural:** ナチュラル
      - <kbd>Tab</kbd> **Lock distances:** 両側に等しい距離でオフセットします。
      - <kbd>D</kbd> **Distance:** マウスの動きで距離を指定します。
5. コマンドダイアログの OK をクリックするか、右クリックして確定します。
