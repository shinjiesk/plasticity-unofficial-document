# Constrained Surface（拘束サーフェイスを作成）

<Reference shortcut="" command="Constrained Surface" button="" />

クリックで指定した点を通るサーフェイスを生成することができます。

## 基本的な使い方

1. コマンドパレットで **Constrained Surface** を実行します。
2. 通したい点をクリックします。
3. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Tolerance:** サーフェスの許容差を指定します。
        - **Angular tolerance:** サーフェスの角度許容差を指定します。
        - **Optimize:**
            - **Performance:** サーフェスを高速に生成します。
            - **Smoothness:** サーフェスを滑らかに生成します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>Cmd-Z</kbd>: 最後に指定した点を取り消します。
4. コマンドダイアログの OK をクリックするか、右クリックで確定します。
