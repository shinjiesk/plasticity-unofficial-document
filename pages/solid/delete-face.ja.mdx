# Delete Face（フェイスを削除する）

<Reference shortcut="Shift + Delete" command="Delete Face" button="" />

フェイスを削除できます。

## 基本的な使い方

1. Select the Faces you want to delete.
1. 削除したいフェイスを選択します。フェイスは複数選択可能です。
2. <kbd>Shift + Delete</kbd>を押します。
3. フェイスが削除されます。
