# Untrim

<Reference shortcut="Alt + T" command="Untrim" button="" />

Create a Sheet based on the original shape of the selected Face.

## Basic Usage

1. Select the Face you want to untrim.
2. Press <kbd>Alt + T</kbd>.
3. A Sheet based on the original shape is created.
