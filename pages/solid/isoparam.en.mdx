# Isoparam

You can Add isoparams (Edges) to a face of a solid or sheet.

1. Press <kbd>Ctrl + R</kbd>.
2. Place the mouse cursor on the position where you want to create the isoparams.
3. Set the options in the Keyboard Shortcuts.
   - <kbd>Tab</kbd> **U/V direction:** Toggle the direction of the isoparam to create.
   - <kbd>Shift</kbd> + Wheel up **Add cut:** Increases the number of isoparams.
   - <kbd>Shift</kbd> + Wheel down **Subtract cut:** Reduces the number of isoparams.
4. Confirm with right-click.
