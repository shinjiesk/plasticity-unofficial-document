# Extend Edge

<Reference shortcut="" command="Extend Edge" button="" />

Verlängern Sie den ausgewählten Edge, bis er einen anderen Edge schneidet.

## Grundlegende Nutzung

1. Wählen Sie die Edges aus, die Sie verlängern möchten.
2. Verwenden Sie den **Extend Edge**-Befehl in der Befehlspalette.
3. Der Edge wird verlängert.
