# Extend

<Reference shortcut="" command="Extend" button="" />

Unified command for **Extend Curve**, **Extend Sheet**. The command is run based on the selected object.
    - When Curves are selected: see [**Extend Curve**](/sketch/extend-curve) in the Sketch Commands section.
    - When outside Edges of the Sheet are selected: see [**Extend Sheet**](extend-sheet).
