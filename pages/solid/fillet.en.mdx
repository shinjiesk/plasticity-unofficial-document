# Fillet

<Reference shortcut="B" command="Fillet" button="" />

Unified command for **Fillet Curve**, **Fillet Vertex**, **Fillet Shell**. The command is run based on the selected object.
    - When Curves are selected: see [**Fillet Curve**](/sketch/fillet-curve) in the Sketch Commands section. .
    - When verticies are selected: see [**Fillet Vertex**](/sketch/fillet-vertex) in the Sketch Commands section. .
    - When Edges are selected: see [**Fillet Shell**](fillet-shell).
