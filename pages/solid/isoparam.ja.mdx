# Isoparam (アイソパラムを加える)

ソリッドとシートのフェイスにアイソパラム（エッジ）を加えることができます。

1. <kbd>Ctrl + R</kbd> を押します。
2. アイソパラムを作成する位置にマウスカーソルを合わせます。
3. キーボードショートカットでオプションを設定します。
   - <kbd>Tab</kbd> **U/V direction:** アイソパラムの方向を切り替えます。
   - <kbd>Shift</kbd> + Wheel up **Add cut:** アイソパラムの数を増やします。
   - <kbd>Shift</kbd> + Wheel down **Subtract cut:** アイソパラムの数を減らします。
4. 右クリックで確定します。
