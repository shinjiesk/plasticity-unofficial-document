# Einführung in Blender Bridge

Die Blender Bridge ist ein Feature, das den schnellen Transfer von Mesh-Daten von Plasticity nach Blender ermöglicht. Dies wird als Blender Add-on realisiert. Sobald Sie verbunden sind, können Sie einfach auf Aktualisieren klicken, und die Daten werden sehr schnell geladen. Sie müssen sich keine Gedanken darüber machen, in eine temporäre Datei zu exportieren und alle Einstellungsoptionen richtig zu bekommen.

Selbst wenn Sie die Geometrie in Plasticity ändern, bleiben die in Blender eingestellten Materialien erhalten. Darüber hinaus wird die Gruppenstruktur in Plasticity als Sammlung in Blender wiedergegeben. In diesem Prozess wird Blenders Hierarchie mit dem Plasticity Dokumentnamen als Identifikator erstellt. Daher können mehrere Plasticity-Dokumente in ein einziges Blender-Dokument integriert werden.

Es gibt auch eine Live-Link-Funktion. Der Live-Link ist ein wirklich cooles Feature; man kann sehen, wie Sachen in Echtzeit aktualisiert werden. Das Hauptarbeitspferd der Blender Bridge ist die Fähigkeit, Assets schnell reinzubekommen, so dass man sie mit Blender Materialien, Blender Beleuchtung, Raytracing sehen kann. Sie können Ihre Objekte in einer Szene mit der echten Kamera betrachten, die möglicherweise in einem Spiel oder in einem finalen Render verwendet wird, mit einem spezifischen FOV. Sie erhalten die volle Leistung von Blender, einschließlich dem Verknüpfen von Dingen zu Leerpunkten, so dass Sie sie animieren, riggen und sehen können, ob Ihre mechanischen Teile zusammenpassen.

Der Workflow ist auch ein fantastisches Erlebnis. Nachdem Sie das Plasticity-Dokument in die Blender-Szene eingefügt haben, können Sie immer noch die Facetteneinstellungen ändern. Fast alle Facetting-Tools von Plasticity sind innerhalb von Blender verfügbar. Das bedeutet, dass Sie schnell die benötigte Polygon-Dichte oder den Edge-Flow für Ihr Modell erhalten können, was zu einem schnelleren Workflow führt.

Die Spezifikation von UV-Nähten und Schärfen in Blender wird in der aktuellen Version nicht beibehalten. Das Blender Add-on kommt jedoch mit Funktionen, die Kanten automatisch markieren, automatisch Geometrie auswählen, die den Plasticity-Flächen oder -Kanten entspricht, und UV-Nähte zusammenführen. Diese Features tragen auch zu einem schnelleren Workflow bei.
