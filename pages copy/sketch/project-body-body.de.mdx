# Project Body Body [#project-body-body]

<Reference shortcut="I" command="Project Body Body" button="" />

Erstellen Sie eine neue Kurve, die die Schnittlinie zwischen zwei Solids oder Sheets darstellt.

1. Wählen Sie zwei Solids oder Sheets aus.
2. Drücken Sie <kbd>I</kbd> oder klicken Sie auf die Schaltfläche.
3. Eine Kurve, die die Schnittlinie darstellt, wird erstellt.
